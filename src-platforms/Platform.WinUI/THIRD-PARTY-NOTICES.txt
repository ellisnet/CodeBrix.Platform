CodeBrix.Platform.WinUI is a family of NuGet packages for WinUI (Windows App
SDK) applications:

  * CodeBrix.Platform.WinUI.ApacheLicenseForever (Core) - the platform-agnostic
    CodeBrix "Simple" MVVM toolkit (SimpleViewModel, SimpleCommand, dialogs,
    messaging, service resolution and OS info) compiled for WinUI.
  * CodeBrix.Platform.WinUI.Skia.ApacheLicenseForever - Skia-rendered image
    controls (EmbeddedImage, EmbeddedImageButton) with vector-direct SVG
    rendering.
  * CodeBrix.Platform.WinUI.Lottie.ApacheLicenseForever - Skia-rendered Lottie
    animation player (AnimatedVisualPlayer, LottieVisualSource,
    ThemableLottieVisualSource).

Jeremy Ellis and contributors license the CodeBrix.Platform.WinUI projects to
you under the terms set forth in the LICENSE file, which is located at the root
of this repository (Apache License, Version 2.0).

CODE PORTED OR DERIVED FROM OTHER PROJECTS
--------------------------------------------------------------------------------

The Skia and Lottie packages contain code ported or derived from the Uno-based
CodeBrix.Platform code elsewhere in this repository, which is itself a fork of
the Uno Platform. Provenance, by area:

1. SVG rendering and image scaling (CodeBrix.Platform.WinUI.Skia)

   The vector-direct SVG rendering in EmbeddedImage, and the stretch/scale math
   in ImageSizeHelper, are ported from the Uno-based CodeBrix.Platform sources
   in this repository:
     - src/AddIns/Platform.UI.Svg (SvgProvider.cs, SvgCanvas.cs)
     - src/Platform.UI/UI/Xaml/Controls/Image/ImageSizeHelper.cs
   Those files were previously part of the Uno Platform (namespaces Uno.UI.Svg
   and Microsoft.UI.Xaml.Controls in the Uno.UI codebase).

   Origin:    Uno Platform - https://github.com/unoplatform/uno
   Copyright: (c) Uno Platform contributors
   License:   Apache License, Version 2.0

2. Lottie player and animated visual sources (CodeBrix.Platform.WinUI.Lottie)

   LottieVisualSourceBase (including the Skottie playback/rendering engine),
   LottieVisualSource, ThemableLottieVisualSource, LottieVisualOptions, and the
   source-driven player contract implemented by this package's
   AnimatedVisualPlayer are ported from the Uno-based CodeBrix.Platform sources
   in this repository:
     - src/AddIns/Platform.UI.Lottie
   Those files were previously part of the Uno Platform (namespace
   Uno.UI.Lottie / CommunityToolkit.WinUI.Lottie type names in the Uno.UI
   codebase).

   Origin:    Uno Platform - https://github.com/unoplatform/uno
   Copyright: (c) Uno Platform contributors
   License:   Apache License, Version 2.0

3. PropertyBindingsParser.cs (CodeBrix.Platform.WinUI.Lottie)

   The CSS-var-style property-binding parser used for Lottie color theming
   came to this repository through the Uno Platform, and originates from the
   Lottie-Windows project.

   Origin:    Lottie-Windows (Windows Community Toolkit) -
              https://github.com/windows-toolkit/Lottie-Windows
              (source/LottieToWinComp/PropertyBindingsParser.cs)
   Copyright: Licensed to the .NET Foundation under one or more agreements.
   License:   MIT License (see text below)

4. System.Json/*.cs (CodeBrix.Platform.WinUI.Lottie)

   A vendored copy (compiled as internal types) of the System.Json mutable
   JSON DOM, used for Lottie color theming because System.Text.Json documents
   are read-only. It came to this repository through the Uno Platform, and
   originates from the .NET corefx repository (the per-file headers record the
   exact corefx export point).

   Origin:    dotnet/corefx - https://github.com/dotnet/corefx
   Copyright: Licensed to the .NET Foundation under one or more agreements.
   License:   MIT License (see text below)
